# invoice_web

SpringBoot+SpringSession+Mysql+React

运行 mvn spring-boot:run

项目启动后 /example 生成测试数据（一个商户JD，编号1-1000购物小票）

/auto 可根据输入手机号码和关联小票自动生成发票，默认会存入用户名：手机号，密码：手机号后四位
